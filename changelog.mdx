<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Enhanced FHIR R4 support: Integrate patient records seamlessly with new endpoints like `/fhir/Patient` and `/fhir/Observation`.
- Automated medical documentation: AI-powered transcription now handles Czech and English clinical notes with `>95%` accuracy.

## Improvements

- Optimized Python SDK for HealthTech workflows, reducing latency by `30%` during FHIR data syncing.
- Next.js dashboard updates with real-time AI inference previews.

## Bug Fixes

- Fixed token refresh issues in long-running sessions.
- Resolved edge cases in FHIR resource validation.

</Update>

## Upgrading to the Latest Version

Follow these steps to upgrade your Sovadina integration.

<Steps>
  <Step title="Review Changelog" icon="book-open">

    Scan the release notes above for breaking changes.

  </Step>
  <Step title="Update SDK" icon="download">

    <CodeGroup tabs="Python,JavaScript">
    ````python
    pip install sovadina==1.2.0
    ````

    ````javascript
    npm install @sovadina/sdk@1.2.0
    ````
    </CodeGroup>

  </Step>
  <Step title="Test FHIR Integration" icon="shield">

    Verify endpoints with your `YOUR_API_KEY`:

    ```bash
    curl -H "Authorization: Bearer YOUR_API_KEY" \
         https://api.example.com/v1/fhir/Patient
    ```

  </Step>
  <Step title="Deploy Changes" icon="rocket">

    Restart your services and monitor at `https://dashboard.example.com`.

  </Step>
</Steps>
