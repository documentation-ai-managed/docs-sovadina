## Core Technologies

Sovadina leverages industry-leading technologies. You integrate Next.js for dynamic frontends, Python for backend AI logic, and advanced models for healthcare-specific tasks.

<Tabs>
  <Tab title="Next.js" icon="code">
    Create responsive dashboards for health data visualization.

````jsx
// pages/dashboard.js
import { useState, useEffect } from 'react';

export default function HealthDashboard() {
  const [data, setData] = useState([]);

  useEffect(() => {
    fetch('https://api.example.com/health-data')
      .then(res => res.json())
      .then(setData);
  }, []);

  return (
    <div>
      {data.map(patient => (
        <div key={patient.id}>{patient.name}</div>
      ))}
    </div>
  );
}
````
  </Tab>
  <Tab title="Python" icon="python">
    Process FHIR resources with AI models.

````python
# fhir_processor.py
import fhirclient
from fhirclient import server

smart = server.FHIRServer('https://api.example.com/fhir')
patient = smart.request_json('Patient/123')

print(f"Patient: {patient['name'][0]['text']}")
````

## Quick Start

Get up and running with Sovadina in minutes. Follow these steps to build your first HealthTech prototype.

<Steps>
  <Step title="Sign Up" icon="user-plus">

  </Step>
  <Step title="Get API Key" icon="key">
    Generate your key from the dashboard settings.

  -H "Authorization: Bearer YOUR_API_KEY"
